== Requirements

Requires http://lucene.apache.org/solr/[Apache Solr] 5. Preferably the latest 5.x version.

[source,xml]
----
<dependency>
  <groupId>org.apache.solr</groupId>
  <artifactId>solr-core</artifactId>
  <version>${solr.version}</version>
</dependency>
----
